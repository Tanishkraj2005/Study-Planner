Study Planner

Study Planner is a responsive web application that helps students organize subjects, manage study schedules, track tasks, 
and monitor progress. Built with HTML, CSS, and JavaScript, it uses LocalStorage to save all data permanently without any backend.

Features

Dashboard Overview – Quick view of subjects and upcoming tasks
Subject Management – Add, delete, and prioritize subjects
Weekly Schedule Planner – Create a study timetable with conflict detection
Task Manager – Add deadline-based tasks and mark them as completed
Progress Analytics – Simple visual charts for completed vs pending tasks
Settings Panel – Dark/Light mode, export data, and clear all data
LocalStorage Support – Persistent data even after refresh or browser restart

Tech Stack

HTML5
CSS3 (Fully responsive + dark mode)
Vanilla JavaScript
LocalStorage API
